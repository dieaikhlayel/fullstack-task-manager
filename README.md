## Full Stack Task Management System

### Tech Stack
- Node.js (Express)
- MongoDB
- React
- Angular
- JWT Authentication

### Features
- User authentication
- Role-based access (User/Admin)
- RESTful API
- Secure endpoints

### Project Structure
- backend/ → API
- frontend-react/ → User UI
- admin-angular/ → Admin dashboard

### How to Run
1. Run backend
2. Run React frontend
3. Run Angular admin panel
